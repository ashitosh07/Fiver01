CSS Flexbox:flex-wrap[1]

with a flexbox layout by default items will try to fit onto one line . You can change that and allow the items to wrap as nedded wit this property

Usage

This property supports the following values:

  • nowrap(default):all flex items will be on one line
  • wrap:flex items will wrap onto multiple lines,from top to bottom
  • wrap-reverse:flex items willl wrap onto multiple lines from bottom to top

Example

.container {
flex-wrap : nowrap | wrap | wrap-reverse;
}

References

Coyier,Chris (Dec 9th,2013) A Complete Guide to Flexbox.CSS Tricks
